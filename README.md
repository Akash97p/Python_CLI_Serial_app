Simple python script to listen/send data to serial ports.
Options
* Scan ports  
* Listen
* Send data
* **store serial data to txt file**.

# Installatig Dependencies

```sh
$ cd SerialCLI
$ pip install -r requirements.txt
```

# Run

```sh
$ cd SerialCLI
$ sudo python3 main.py
```
